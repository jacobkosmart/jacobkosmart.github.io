# 1.상수와 변수

- 상수는 변하지 않는 일정한 값

- 변수는 변할 수 있는 값을 갖는것

```swift
import Foundation

// 상수
// let 상수명: 데이터 타입 = 값

let a: Int = 100


// 변수
// var 변수명: 데이터 타입 = 값
var b: Int = 200
b = 400

```

# 2. Swift 기본 데이터 타입

- Int: 64bit 정수가

- UInt: 부호가 없는 64bit 정수형

- Float: 32bit 부동 소수점

- Double: 64bit 부동 소수점

- Bool: true, false 값

- Character: 문자

- String: 문자열

- Any: 모든 타입을 지정하는 키워드

```swift
import Foundation

// Int
var someInt: Int = -100
someInt = 100

// UInt
var someUInt: UInt = 200

// Float
var someFloat: Float = 1.1
someFloat = 1
print(someFloat)

// Double
var someDouble: Double = 1.1
someDouble = 1

// Bool
var someBool: Bool = true
someBool = false

// Character
var someCharacter: Character = "가"
someCharacter = "A"
someCharacter = "😄"

// String
var someString: String = "안녕하세요 👉"

// 타입추론
var number = 1
```
