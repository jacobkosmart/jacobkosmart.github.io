물품 교환 플랫폼 서비스
## 프로젝트 소개  
### 팀 구성  
 3인 팀 프로젝트
### 사용기술 및 언어    
  Java 1.8  
  React 16.9.46  
  MYSQL 8.0  

### 개발 기간  
2020-08-03 ~ 2020-09-05(약 5주)

### 특징  
100% 비대면 진행 프로젝트

### 주요 기능
 - 사진과 함께 교환을 원하는 물건 등록  
 - 실시간 채팅으로 물건 판매자와 대화  
 - 지정한 주소에 원하는 물건 검색
